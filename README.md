
# @melody-core/leo

狮子座是旋律之主们基于commander开发的脚手架工具，它的能力来源于它集成的模板。它既可以是独立的命令行工具，也可以作为@melody-core/melody-cli的一个命令行套件。


## install && use

```shell
    # 你可以单独安装并使用它
    sudo yarn global add @melody-core/leo
    # use
    leo



    # or 将其作为melody-cli的一个套件, 我们更期望如此。
    melody install @melody-core/leo
    # use
    melody leo
```
